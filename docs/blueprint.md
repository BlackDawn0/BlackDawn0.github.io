# **App Name**: Dev Hub

## Core Features:

- Home Page: Landing page with a brief introduction, asset list, game list, and a small contact section.
- Contact Page: Form submission page.
- Asset List: Displays a curated list of programming assets with descriptions and links on the Home page.
- Game List: Displays a list of programming-related games with descriptions and links on the Home page.
- Contact Section: Contact form submission section at the bottom of the Home page

## Style Guidelines:

- Background color: Dark gray (#282A36) for a dark mode theme.
- Primary color: Light purple (#BD93F9) to add vibrancy and focus in a dark UI.
- Accent color: Teal (#8fcfb4) for interactive elements and important highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern look that's suitable for both headlines and body text.
- Use minimalistic icons to represent different categories and functionalities.
- Clean, structured layout with a focus on readability and easy navigation.
- Subtle transitions and animations for a smooth user experience.
